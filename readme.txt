== Description ==

The main idea of the plugin – make your site content is searchable by meta fields and taxonomies.

The plugin has very high flexibility and for now it is the most comprehensive solution for making content 
is searchable by meta fields AND taxonomies among wordpress plugins on codecanyon and wordpress.org!


With this plugin you will be able to filter posts/pages/custom types by the meta fields and/or 1 taxonomy.


* You want to make searchable – car’s, homes for sale,rent,hotels,ships, wooshop items and any other things on your site – the plugin is for you
* You are Real Estate agency – the plugin is for you
* You are Car Dealer – the plugin is for you
* You have Culinary site with tons of recipes – the plugin is for you
* Your site is a Showcase or your Portfolio – the plugin is for you
* Your site is WooCommerce shop and you want to make products are searchable by meta fields and taxonomies – the plugin is for you
* If your site has metafields already, and you want make them searchable – the plugin is for you!


https://www.youtube.com/watch?v=o_jFjWFVdZY


FREE FEATURES:

    * Create new meta fields with built-in constructor (checkbox, drop-down, range-slider)
    * Make searchable your products by new meta fields and 1 taxonomy
    * qTranslate ready
    * Auto submit (for meta fileds)
    * Reset button


CODECANYON VERSION FEATURES:

    * AJAX RECOUNT OF ITEMS WITHOUT PAGE RELOADING
    * Create new meta fields with built-in constructor (checkbox, drop-down, range-slider)
    * Filter posts,products etc. by meta fields AND multiple taxonomies in the same time!
    * Make searchable your products by already existing meta fields and taxonomies!
    * Make searchable your products by new meta fields and taxonomies
    * INSERT SEARCH FORMS BY SHORTCODES WHERE YOU WANT
    (you will maybe need css skin to insert search form in some places of your site, this feature supports as customization for some extra fee)
    * CREATE SHORTCODES BY the CONSTRUCTOR
    * CREATE SHORTCODES SKINS AND SET IT FOR EACH SHORTCODE
    * Marketing short SEO-fiendly links service to promote your items
    (Example: http://meta-data-filter-demo2.pluginus.net/woman-dresses/175/germany-bonn-woman-dresses-for-all-seasons-of-year)
    * Taxonomy values auto recount based on the current search
    * The 'Reflection' functionality for metafields for working with already existed metafields of your site
    Reflections – is functionality for synchronizing new meta fields with already existing meta fields and makes them searchable!
    * Each widget AND shorcode has its own customizable constructor of order-by panel
    * Reset button optionally
    * qTranslate ready
    * Steps optionally for a range slider
    * IT IS NOT JUST A PLUGIN IT IS THE INFO COMBINE
